ASRA starts as a **general adaptive reasoning architecture** first.

It becomes strongly and explicitly about **Decision Biology** beginning in:

# Phase 8 — Decision Biology Bridge

That is the first phase where:

```text
environment → biological system
action → perturbation
state → cellular state
goal inference → survival/adaptation objective
transition graph → signaling dynamics
```

becomes the central framing rather than just an analogy.

But philosophically, the transition actually happens gradually across multiple phases.

Here’s the deeper breakdown:

```text
PHASE 1–3
ASRA = General Adaptive Intelligence System

Focus:
- exploration
- abstraction
- memory
- symbolic reasoning
- planning
- unknown environments

Primary domains:
- ARC-AGI-3
- MiniGrid
- ARC
- Procgen

This stage builds:
“the cognitive substrate”
```

```text
PHASE 4–5
ASRA begins to resemble Decision Biology conceptually

Why?

Because ASRA starts learning:
- action semantics
- causal inference
- hidden-state reasoning
- hypothesis generation
- adaptive response modeling

This is already biologically meaningful.

At this point:

environment state → action → next state

starts looking very similar to:

cell state → perturbation → next cell state
```

This is the conceptual bridge phase.

PHYRE, CLEVRER, and transition modeling are especially important here because they introduce:

* causality
* intervention reasoning
* counterfactuals
* state propagation

which are core to Decision Biology.

---

Then comes the major shift:

```text
PHASE 6–7
ASRA becomes a mature adaptive scientific reasoning system
```

At this point ASRA already has:

* memory
* planning
* hypothesis testing
* causal modeling
* strategy transfer
* world modeling

Now the architecture is no longer tied to games.

It becomes a reusable scientific reasoning engine.

This is the critical transition point.

---

Then:

```text
PHASE 8
ASRA becomes explicitly Decision Biology
```

because now the datasets themselves are biological:

* LINCS L1000
* scPerturb
* OmniPath
* Cell Painting
* Human Cell Atlas

and ASRA is no longer solving:

```text
grid-world reasoning
```

but instead:

```text
cellular adaptive reasoning
```

At this phase, ASRA starts doing things like:

```text
infer hidden signaling mechanisms
predict cellular responses
reason over perturbations
model adaptive pathways
discover latent biological states
generate pathway hypotheses
```

This is where:

# ASRA → Decision Biology Engine

happens.

---

The most important insight is this:

```text
Decision Biology is NOT a separate architecture from ASRA.
```

Instead:

```text
Decision Biology is a domain specialization of ASRA.
```

Meaning:

```text
ASRA = general adaptive scientific reasoning architecture

Decision Biology = ASRA applied to cellular systems
```

Exactly like:

```text
a physics engine can simulate:
- rigid bodies
- fluids
- cloth
- particles

using the same core mechanics
```

ASRA becomes the core reasoning engine,
while Decision Biology becomes one of its scientific domains.

---

The ideal long-term evolution is probably:

```text
ASRA Core
    ↓
Scientific World Modeling
    ↓
Decision Biology
    ↓
Nature Foundation Models
```

Where:

```text
ASRA
= adaptive reasoning engine

Decision Biology
= first scientific domain

Nature Foundation Models
= multi-domain scientific intelligence platform
```

That hierarchy is extremely strong conceptually because it cleanly separates:

```text
reasoning engine
vs
scientific application layer
vs
foundation-model ecosystem
```
