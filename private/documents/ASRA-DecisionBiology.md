# ASRA + Information Theory

This is where your work becomes especially differentiated.

Decision Biology already uses:

* Mutual Information,
* Channel Capacity,
* Information Loss,
* Signaling Fidelity. 

ASRA can use those directly as reasoning objectives.

For example:

The planner can choose experiments that maximize information gain.

This connects directly to information theory.

For uncertainty reduction:
I(X;Y)=\sum_{x\in X}\sum_{y\in Y}p(x,y)\log\frac{p(x,y)}{p(x)p(y)}

For channel capacity:
C=\max_{p(x)} I(X;Y)

This is extremely powerful because it transforms biology into:

```text
information flow optimization
+
adaptive scientific reasoning
```

Very few AI biology systems think this way.
