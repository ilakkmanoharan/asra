## Title

Understanding Action Semantics Inference Through State Transitions in ASRA

## Description

A ~7.5-minute educational video explaining how ASRA discovers what actions mean—without ever being told. Using a simple 3×3 grid example, it walks through Before State → ACTION1 → After State, change detection, semantic convergence (0→1→2→3→0), and the formula `next[y][x] = (grid[y][x] + 1) % 4`, then connects the idea to ARC reasoning, world models, Decision Biology, and the ASRA adaptive reasoning loop.
